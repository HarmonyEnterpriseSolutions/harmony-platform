Для прямого запуска использования форм (мимо login.gfd)

нужно запустить 

link_default_app_configs.py <имя приложения>

Этот скрипт создает HARD LINKS файлов

connections.conf
fn_signatures.conf.py

на соответствующие файлы в папке приложения

Редактирование этих файлов в папке etc повлечет изменения линкованных файлов.

Внимание: после апдейта оригинальных файлов связь разрывается!
          необходимо перелинковать
          запуск скрипта без параметров удаляет линки
