5. Устанавливаем апач
	
	I:\dist\apps\apache\httpd-2.0.64-win32-x86-no_ssl.msi 

	Папка wm/www должна быть доступна как http://localhost/wm/
	Скрипты wm/www/*.cgi должны быть доступны на выполнение, остальное на чтение

	Добавить в httpd.conf
	-----------------------
	Alias /wm/ "C:/Programs/wm/www/"

	<Directory "C:/Programs/wm/www">
		Options ExecCGI Indexes MultiViews FollowSymLinks
		AllowOverride None
		Order allow,deny
		Allow from all
		AddHandler cgi-script .cgi
	</Directory>
	-----------------------

	Проверка
		http://localhost/wm/wk.cgi/ должна давать страницу вебкита, значит вебкит правильно связан с апачем
		http://localhost/wm/wk.cgi/wm/info/ должна давать страницу с заголовком "Request stuff:" значит контекст вебкита wm настроен верно

	cd C:\Programs\wm\bin
	редактируем первую строчку config.py
		server_url = 'http://localhost/wm/wk.cgi/wm'

	Проверка 
		скрипт wm.py дает диалог логина, но дальше не идет, т.к. нет базы
